From InterCatch to CEF age–structured data for sole 8ab

JB Lecomte

2026-06-23

Overview

This vignette illustrates the workflow used to transform InterCatch data into age–structured CEF data for sole in ICES areas 8ab, including:
- Importing InterCatch data and converting to CEF format.
- Linking CEF catches and distributions via domainBiology.
- Applying age allocation to strata without biological sampling.
- Producing summary tables and diagnostic plots for quality control.

Raised catch overview and strata definition

We load the raised landings and discards overview, then standardise strata definitions using cef_remove_strata() and cef_add_strata().
The seasonValue is coerced to character to ensure consistent joins downstream.

df_stock_overview_L_D_raised <- readRDS(
  here(glue("WGRDBESstockCoord/personal/JB/output/df_stock_overview_L_D_raised{data_year}.rds"))
) %>%
  cef_remove_strata() %>%
  cef_add_strata() %>%
  mutate(seasonValue = as.character(seasonValue))

Link catches and distributions via domainBiology

We extract unique combinations of domainBiology, catchCategory, and strata_full that are needed to link CEF catches and distributions.
Catch categories are harmonised (e.g. "Dis_total" → "Dis") to match the categories used in the CEF distribution table.

df_catch_domainBiology <- df_stock_overview_L_D_raised %>%
  filter(!is.na(domainBiology)) %>%
  select(
    domainBiology,
    catchCategory,
    strata_full
  ) %>%
  mutate(
    catchCategory = case_when(
      catchCategory == "Dis_total" ~ "Dis",
      catchCategory == "BMS_total" ~ "BMS",
      catchCategory == "Lan_total" ~ "Lan",
      TRUE                         ~ catchCategory
    )
  ) %>%
  distinct()

CEF distributions and merging with catches

We start from the CEF distributions table, harmonise numeric values and catch categories, then link to the catch strata via domainBiology.
Strata are expanded using cef_expand_strata_full() to ensure all derived strata fields are available.

cef_distribution <- list_cef$distributions %>%
  mutate(
    value         = as.numeric(value),
    catchCategory = case_when(
      catchCategory == "DIS" ~ "Dis",
      catchCategory == "BMS" ~ "BMS",
      catchCategory == "LAN" ~ "Lan",
      TRUE                  ~ catchCategory
    )
  ) %>%
  left_join(
    df_catch_domainBiology,
    by = join_by(catchCategory, domainBiology)
  ) %>%
  mutate(
    importedOrRaised = "imported",
    Age              = distributionClass
  ) %>%
  droplevels() %>%
  cef_expand_strata_full()

We then pivot the distribution data to obtain age–specific numbers and weights by strata and catch category.

df_canum_imported <- cef_distribution %>%
  pivot_wider(
    values_from = value,
    id_cols     = c(strata_full, distributionClass, catchCategory),
    names_from  = c(variableType, valueType),
    names_sep   = "_",
    unused_fn   = dplyr::first
  ) %>%
  cef_expand_strata_full()

Identify strata with and without age sampling

We distinguish between strata with available age sampling (df_ref_fleets) and strata where age must be allocated (df_allocation_fleets).
The vector strata_no_age keeps track of strata still lacking age information throughout the iterative allocation.

# Reference strata: have imported age information
df_ref_fleets <- df_stock_overview_L_D_raised %>%
  filter(strata_full %in% unique(df_canum_imported$strata_full)) %>%
  droplevels()

# Strata requiring age allocation
df_allocation_fleets <- df_stock_overview_L_D_raised %>%
  filter(!(strata_full %in% unique(df_canum_imported$strata_full))) %>%
  droplevels()

# Strata identifiers without age information
strata_no_age <- df_allocation_fleets %>%
  select(strata_full)

# Reference CEF age distributions (after SOP corrections)
df_canum_for_allocation <- df_canum_imported_sopr

Age allocation rules and iterative allocation

We define a hierarchy of strata–grouping rules to allocate age compositions to unsampled strata.
The allocation proceeds from the most specific grouping to the most aggregated, updating the list of unsatisfied strata after each step.

allocation_specs <- list(
  list(
    group_strata  = "strata_c_q_g",
    join_by_cols  = c("strata_c_q_g", "catchCategory"),
    source_name   = "strata_c_q_g"
  ),
  list(
    group_strata  = "strata_q_g",
    join_by_cols  = c("strata_q_g", "catchCategory"),
    source_name   = "strata_q_g"
  ),
  list(
    group_strata  = "strata_q_sg",
    join_by_cols  = c("strata_q_sg", "catchCategory"),
    source_name   = "strata_q_sg"
  ),
  list(
    group_strata  = "strata_g",
    join_by_cols  = c("strata_g", "catchCategory"),
    source_name   = "strata_g"
  ),
  list(
    group_strata  = "strata_sg",
    join_by_cols  = c("strata_sg", "catchCategory"),
    source_name   = "strata_sg"
  ),
  list(
    group_strata  = "strata_c_q",
    join_by_cols  = c("strata_c_q", "catchCategory"),
    source_name   = "strata_c_q",
    extra_filter  = quote(gear == "MIS") # optional extra filter if used in helper
  ),
  list(
    group_strata  = "strata_c",
    join_by_cols  = c("strata_c", "catchCategory"),
    source_name   = "strata_c"
  )
)

We then loop over the specification list and apply cef_allocate_strata() at each level.
After each allocation step, we update strata_no_age so that remaining strata are passed to the next, more aggregated, rule.

allocated_list <- list()

for (i in seq_along(allocation_specs)) {
  spec <- allocation_specs[[i]]

  allocated_list[[i]] <- cef_allocate_strata(
    df_allocation_fleets    = df_allocation_fleets,
    df_canum_for_allocation = df_canum_for_allocation,
    strata_no_age           = strata_no_age,
    number_col              = "Number_Total_SopR",
    group_strata            = spec$group_strata,
    join_by_cols            = spec$join_by_cols,
    source_name             = spec$source_name
  )

  # Remove strata that have just been allocated from the remaining pool
  strata_no_age <- strata_no_age %>%
    filter(!(strata_full %in% levels(allocated_list[[i]]$strata_full))) %>%
    droplevels()
}

Combine imported and allocated CEF age data

We bind all allocated strata, clean up temporary variables, and recombine them with the original imported age data.
Weights and total catch columns are standardised, and strata fields are flattened for later aggregation and export.

df_canum_allocated <- bind_rows(allocated_list) %>%
  cef_remove_strata() %>%
  rename(
    Caton   = Catchkg,
    weights = weights_ref
  ) %>%
  select(
    -any_of(c(
      "DiscardsImportedOrRaised",
      "raised_landings",
      "numbers_ref_tot",
      "weca_tot",
      "SoP"
    ))
  ) %>%
  mutate(seasonValue = as.character(seasonValue))

df_canum_for_allocation <- df_canum_for_allocation %>%
  cef_remove_strata()

df_canum <- bind_rows(
  df_canum_for_allocation,
  df_canum_allocated
) %>%
  cef_remove_strata()

saveRDS(
  df_canum,
  file = glue("{path_data_raised}/df_canum_all_age_{data_year}.rds")
)

Age–plus grouping

For assessment purposes we aggregate all ages at or above max_agestock into a single plus group.
Numbers are summed by strata_full, Age, and catchCategory.

df_canum_ageplus <- df_canum %>%
  mutate(
    Age = ifelse(
      as.numeric(Age) >= as.numeric(max_agestock),
      as.character(max_agestock),
      Age
    )
  ) %>%
  group_by(strata_full, Age, catchCategory) %>%
  mutate(Number_Total = sum(Number_Total)) %>%
  distinct()

saveRDS(
